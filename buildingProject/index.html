Собираем проект: 

$ bem make

Первая сборка с bem-tools займет некоторое время, так как именно в этот
момент в директорию проекта устанавливается все необходимые npm-пакеты.

Для запуска сервера выполняем: 

$ bem server

В результате вы увидите следущее событие

info: Server is listening on port 8080. Point your browser to http://localhost:8080.domain

Проблема?

Если порт 8080 уже используется другой программой, его можно переназначить с
помощью аргумента -p.
